# Portfolio Website

A modern, responsive portfolio website built with **React 18**, **Vite**, **Tailwind CSS v4**, and **Framer Motion**. Features a clean UI with dark/light theme support, interactive animations, GitHub stats integration, and a contact form powered by EmailJS.

## 🚀 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **3D Effects**: Three.js / @react-three/fiber
- **Icons**: Lucide React, React Icons
- **Contact Form**: EmailJS
- **Deployment**: Vercel / Netlify

## ✨ Features

- 🌓 Dark/Light theme toggle
- ⚡ Smooth page transitions & scroll animations
- 🎯 Interactive particle background
- 📊 Live GitHub stats integration
- 📱 Fully responsive design
- ♿ Reduced motion support
- 🔍 SEO optimized with Open Graph & JSON-LD
- 📬 Contact form with EmailJS

## 🛠️ Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/jayavgune18/portfolio-website.git

# Navigate to the project
cd portfolio-website

# Install root dependencies
npm install

# Navigate to front-end and install dependencies
cd front-end
npm install

# Start the development server
npm run dev
```

### Build for Production

```bash
cd front-end
npm run build
```
